# dot

Easily manage dotfiles through the command line.


# Installation

Simply run:

    $ pip install dot
    
Or if you want to install it from source
    
    $ git clone https://github.com/kylefrost/dot.git
    $ cd dot/
    $ python setup.py install


# Usage

To use it:

    $ dot --help
